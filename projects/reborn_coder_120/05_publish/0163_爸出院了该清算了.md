# 第163章 爸出院了该清算了

晨风吹过临江市人民医院住院部的香樟树，落了半肩细碎的阳光。苏哲拎着换洗衣物推开病房门的时候，苏振海已经自己收拾好了帆布包，正站在窗台边看楼下的车水马龙，背挺得笔直，完全不是三个月前躺在床上气若游丝的样子。床头柜上放着半颗削好的苹果，果皮刀摆得整整齐齐，看得出来是他自己动手收拾的。

“都说了我来收拾，您坐着歇着。”苏哲放下东西过去接包，指尖碰到苏振海的手腕，能感觉到扎实的力度，不是之前的绵软。

“心结开了，身子骨就跟着松快，哪用歇着。”苏振海拍了拍苏哲的肩膀，力道不轻不重，带着失而复得的踏实，“昨天你说张广涛的判决下来了？我昨晚翻了法院的公开公告，只看到了判决结果，没看到详情，具体怎么说的？”

苏哲扶着他往门口走，一边走一边说：“昨天终审出的结果，老赵利用职务便利协助转移资产，五年有期徒刑，追缴全部违法所得，一分都没跑了。张广涛是主犯，职务侵占、故意调换体检报告延误治疗，数罪并罚判了七年，退赔云帆两百八十万，罚金一百万。”

这话落进苏振海耳朵里，他脚步顿了顿，抬手抹了把眼角，半天叹了口气：“八年了，从被抓进去到现在，整整八年，终于落定了。”

出了住院部办手续，苏哲开车往云帆科技方向走，路过滨江路的时候，苏振海忽然开口：“你跟我说实话，回去之后，你打算怎么安排我？我知道现在云帆是你撑着，那个LLM辅助开发框架也是你一手攒出来的，我不想抢你位置，也坐不动CEO的位置了，就是闲不住，想帮着干点活。那些跟着我打天下的老兄弟，好多这些年被张广涛挤兑得难受，我去跟他们聊聊，也能稳住人心。还有公司合规，之前张广涛把这块搅得乌烟瘴气，我去理，你专心盯技术就行。”

苏哲从后视镜里看了他一眼，笑了：“我本来就是这么打算的。让您当云帆的监事，兼合规委员会主席，老员工对接全归您，不用天天坐班，有事您来，没事您回家养花钓鱼，多好。我专心盯LLM项目和战略，刚好不用分心管这些烂事。”

苏振海听完咧嘴笑了，连说这个安排合适，他满意。

车子拐过两个路口就到了市中级人民法院，陈曦早已经抱着整理好的材料袋在门口等，全黑的文件袋封得整整齐齐，边角都压得平平整整。苏哲让苏振海在门口咖啡馆坐着歇会，自己跟着陈曦进去交材料。

“所有证据都核对了三遍，当年张广涛挪用资产转到蓝星空壳公司的银行流水，第三方评估机构做出来的入股估值造假鉴定，还有张广涛判决书里认下的口供，全齐了，刚好对应1.2亿的不当得利，一分不差。”陈曦一边走一边跟苏哲说，声音压得低，“立案庭昨天打过招呼，说材料齐的话今天就能收。”

果然，不到二十分钟，材料就全部核验签收，立案庭给了回执，说七个工作日内出是否受理的通知。苏哲把回执揣进包里，和陈曦一起往外走，刚下到法院门口的台阶，就看见王堔靠在黑色奔驰的车头，手里夹着烟，正抬着眼看他们。

“巧了，我就是来等苏总的。”王堔把烟掐灭在路边的烟灰缸里，拍了拍手，“蓝星那边关于LLM项目启动会旁听的事，我带了准信过来。”

苏哲靠在台阶的栏杆上，抱着胳膊笑：“我的规则之前说过了，所有旁听人员必须签保密协议，泄密赔三倍项目估值，会场不准带任何电子设备，入口过安检，不管是谁都一样，没的谈。”

王堔皱了皱眉：“苏哲你这太过分了，我们毕竟是云帆登记在册的股东，看一眼自己投的项目怎么了？用得着防贼一样？”

“防的就是贼。”苏哲语气没松，“当年你们拿张广涛转的1.2亿不当入股，把云帆原本的现金流掏空，逼得我爸背了八年黑锅，现在还有脸说自己是正经股东？今天我刚把追讨材料交了，法院已经收了，等把这1.2亿追回来，你们的股权本来就该按比例调，现在想沾核心技术的光，门都没有。”

“好了好了，谈条件就谈条件，说这些气话干什么。”王堔抬了抬手，示意他别激动，“蓝星同意你的规则，保密协议我们签，安检我们也认，就一个条件——你把这个立案撤了，咱们坐下来谈，蓝星给LLM项目追加一轮投资，估值给你翻一倍，大家都有钱赚，何必闹得这么僵？”

话音刚落，身后就传来苏振海的声音，他刚才喝完茶出来透气，一步步走到台阶上，声音清亮得很：“王堔，当年我跟蓝星签入股协议的时候，记得有一条写得清清楚楚，投资方违规干预公司核心经营，串通高管侵占公司资产的，创始人团队有权按原始入股价回购投资方全部股权，你还记得这一条吗？”

王堔的脸“唰”一下就白了，他盯着苏振海看了半天，才哑着嗓子说：“那都多少年前的老条款了，你居然还记着？”

“我自己签的字，我能忘？”苏振海笑了，“当年张广涛管档案，我怕他动歪心思把原件毁了，特意把原件藏在老家老房子的房梁上，上个月苏哲已经拿出来给律师看过了，条款合法有效。现在张广涛的判决都下来了，串通蓝星转移资产已经坐实，条款早就触发了，我们不光追那1.2亿的不当得利，还能直接回购你们全部股权，把你们清出云帆，你说我们为什么要撤立案？”

王堔的脸一下子沉得能滴出水，他咬了咬牙：“苏振海，苏哲，你们别逼得太狠，蓝星的体量摆在这，真闹到鱼死网破，你们一个不到百人的小公司，扛得起吗？”

“扛不扛得起，试过才知道。”苏哲往前走了一步，挡在苏振海身侧，“当年你们把我爸逼得含冤入狱，调换体检报告延误治疗，差点把命搭进去，把云帆折腾得快散架的时候，怎么不说逼得太狠？现在我们只是拿回属于我们自己的东西，就成我们不对了？”

王堔盯着俩人看了半天，最后哼了一声：“行，我把话带回去，保密协议我们签，明天我带蓝星派的人去云帆签，规则我们认。但你们别后悔。”说完拉开车门踩了油门走了，轮胎碾过路牙带起一阵灰尘。

陈曦皱着眉问：“这个回购条款真的能生效？我之前整理档案的时候完全没找到，原来真藏了这么一手？”

“那时候我刚创业，防人之心不能没有，没想到真用上了。”苏振海笑了笑，拍了拍苏哲的胳膊，“走吧，回公司，把会场的事定下来。”

一行人开车到云帆科技，一楼前台的小姑娘看见苏振海，惊讶得站起来打招呼，不少老员工从办公室出来，围过来跟苏振海握手，七嘴八舌问身体情况，苏振海笑着一一回应，精神头好得看不出是刚出院的人。

进了会议室，林溪早已经把做好的会场管控方案打出来放在桌上，她推了推眼镜说：“流程都捋好了，入口放安检机，所有电子设备统一寄存，旁听席跟演示台隔了三米，不准靠近拍照，保密协议我们法务已经改了三版，每一条都卡死了，签完按手印留底，只要泄密直接起诉，跑不了。明天蓝星那边来人，直接按这个走就行。”

苏振海坐到大沙发上，清了清嗓子说：“我刚才路上也跟苏哲说好了，我回来之后就当监事，管合规和老员工对接，公司大小事务还是苏哲拍板，LLM辅助这个项目是苏哲一手做起来的，是云帆未来的命根子，谁要是拖后腿，不管是谁，我第一个不答应。”

林溪带头鼓掌，陈曦也跟着笑，会议室里的气氛一下子松快下来。苏哲拿起桌上的法院回执，摸了摸纸面，抬眼看向几个人：“材料交了，规则定了，人也回来了，明天签完保密协议，启动会准期开。开完会，咱们就等着开庭，把最后该拿的，都拿回来。”

他话音刚落，手机震了一下，点开是法院发来的短信，材料已经正式受理，案号都已经生成了。苏哲勾了勾嘴角，把手机扣在了桌上。
