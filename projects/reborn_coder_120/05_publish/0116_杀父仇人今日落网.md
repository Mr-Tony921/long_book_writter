# 第116章 杀父仇人今日落网

货车后斗的帆布被临江的夜风掀得哗哗响，苏哲攥着那枚磨得发亮的铜柄钥匙，指节压得泛白。陈曦抱着打开的笔记本坐在他身侧，指尖飞快敲着触控板，屏幕上跳着张广涛传来的外围布控点位图，红色圈已经把整栋旧仓库围得密不透风。

“王堔昨天从临时管控区脱逃的方向就是这儿，他肯定盯着这块肥肉很久了。”陈曦把屏幕转向苏哲，“我把你攒出来的LLM辅助分析工具连了外围的热成像，只要有活物进了仓库范围，三秒钟就能锁定位置，误差不超过半米。”

苏哲点头，指尖蹭过钥匙上刻得极小的云帆科技logo，心头翻涌着上辈子临死前的记忆——十年前他就是在这栋废弃仓库里，被王堔从承重梁上推下来，摔断了脊椎，眼睁睁看着对方拿走了苏振海藏的东西，到死都没等到翻案的那天。重生回到2019年，他改了怯懦性子，攒了一年多的劲，终于一步步走到这一步了。

卷闸门被苏哲拉开半扇，灰尘裹着旧机器的霉味涌出来，两人摸黑进去，摸到墙角的电闸，啪的一声按下去，昏黄的灯管晃了三晃，终于亮起来，照亮满屋子堆得密密麻麻的淘汰服务器。苏哲顺着上辈子残留的记忆往最里面走，果然在一堆落满灰的机柜后面，找到了那个刷着褪成天蓝色漆的铁皮保险柜，锁孔的大小刚好对上手里的铜钥匙。

咔哒一声脆响，保险柜弹开一道缝，苏哲刚伸手去拉柜门，身后就传来了手枪保险打开的轻响。

“挺会找啊，苏哲，不愧是苏振海的种。”王堔从服务器堆后面转出来，烟夹在指缝里烧了半根，烟灰落在他皱巴巴的名牌西装上，“我等你三个小时了，只要把硬盘和全量用户数据给我，我给你个痛快，怎么样？”

苏哲没回头，反而把柜门完全拉开，露出里面用防水袋层层包着的四块硬盘，忽然笑出声：“你真觉得那注销十年手机号发出来的短信，是苏振海发错了，引我来给你带路？”

王堔捏着枪的手紧了紧，愣在原地。

“当年我爸注销那个号码的时候，就跟运营商签了特殊协议，触发条件就是这个保险柜被打开。”苏哲把防水袋拎出来递给陈曦，“说白了，这就是十年前就布好的钓鱼局，钓的就是你这条躲了十年的鱼。你真以为我没发现，你从脱逃那一刻起，就一直跟在我们屁股后面？”

陈曦接过硬盘，拆开防水袋接上笔记本，很快就做完了校验，屏幕上跳出来的哈希值和LLM工具预存的完全匹配，她抬头比了个ok的手势：“全量数据没问题，除了用户信息，还有额外的跨境转账记录，云帆美国总部的流水全在里面。”

王堔脸色瞬间变得惨白，抬枪就要冲过来抢硬盘，仓库门口突然传来尖锐的警哨声，张广涛带着专案组的人冲进来，没等王堔扣动扳机，就把他结结实实按在了满是灰尘的水泥地上，反手拷上了冰凉的手铐。

王堔趴在地上破口大骂，把苏振江和云帆的勾当骂得一清二楚，全被张广涛带去的执法记录仪录得明明白白。苏哲蹲下来，晃了晃手里的铜钥匙，语气冷得像临江冬天的风：“你勾结云帆这么多年，害死我爸，倒卖国内用户数据分赃，今天算是收网了。”

押走王堔的时候，陈曦解码了硬盘里最后一份加密文件，LLM辅助破译完成后，屏幕上跳出来一张手填的码头停泊单，落款时间就是今天深夜——苏振江已经买通了走私船的船家，今晚就要从临江郊野码头跑路出境。

苏哲把密封好的硬盘物证袋扣好，揣进怀里，冲张广涛抬了抬下巴，指尖还留着铜钥匙的冷意：“王堔落网，证据全了，走，下一站，码头抓苏振江。”

铜柄钥匙顺着锁孔滑进去，只听咔哒一声轻响，十年积灰的锁芯应声弹开。苏哲攥住柜门的生锈拉手，一把拽开，浓得呛人的尘雾扑出来，陈曦捂着嘴偏头咳嗽，指尖仍没松开握着的麻醉枪。

尘雾散了些，两人看清柜里的东西——除了一个裹着防潮膜的加密硬盘，只有半张压在底下的泛黄信笺，是苏振海遒劲的钢笔字：“留予我儿苏哲，王堔是内鬼，云帆科技转赃账户在硬盘第三分区。”

头顶晃悠的灯管突然闪了两下，灭掉半幅，仓库深处的阴影里突然传来一声冷笑，王堔拎着一根锈钢管慢慢走出来，鞋踩在碎服务器壳上咔嚓响。

“不愧是苏振海的种，比你老子命还硬，还真找着这了。”王堔舔了舔干裂的嘴角，“把硬盘留下，我给你个全尸，不然咱们今天就在这烂一块。”

苏哲没动，反而把硬盘揣进自己怀里，靠着保险柜门框笑：“你真以为我是单枪匹马过来抢东西？陈曦带的LLM辅助分析工具连了外围热成像，你二十分钟前翻围墙进来的时候，坐标早就发给张广涛了。”

王堔脸色瞬间变了，攥着钢管的手青筋暴起：“你诈我？”

话音刚落，卷闸门方向突然传来脚步声，张广涛的声音冷生生砸进来：“王堔，我们已经封了所有出口，放下武器投降。”

王堔红了眼，嘶吼着拎着钢管就冲过来，苏哲早有防备侧身一躲，王堔收不住劲撞在保险柜角上，闷哼一声，张广涛带的专案队员已经冲上来，三下五除二把他按在地上，冰凉的手铐咔哒锁上，直接架了出去。王堔挣扎着骂，声音越来越远，彻底没了动静。

这是这辈子苏哲第一次亲手把杀父仇人送进去，他指尖蹭着怀里的硬盘，喉咙发紧——重生回2019年，他从一文不名的普通职员走到今天，终于讨到了第一笔血债。

陈曦把硬盘插进笔记本，先翻出了一个旧网关配置文件，刚好解开了苏哲疑惑很久的短信谜题：注销十年的手机号能发信，是苏振海十年前就设置好的定时触发程序，绑定了专案组立案的时间点自动发送，发信人就是苏振海提前布好的后手，身份彻底坐实。

随后她启动LLM辅助破解，只用了八分钟就打开了第三分区，里面完完整整躺着云帆科技当年倒卖的全量用户数据，还有王堔十年来收受贿赂、配合转赃的全部流水，每一笔都标注得清清楚楚。可就在工具扫完所有可见分区的时候，屏幕突然弹出隐藏分区提示，解锁之后，只有一段一分钟的加密语音，还有一张今早刚出的登机牌截图——登机人赫然是苏振江，目的地纽约，起飞时间就是今天下午三点。

现在临江市的时间，刚过两点十分。苏哲抬头看向陈曦，窗外的警笛声刚好顺着卷闸门的缝隙飘进来，而跨境协查的申请，刚写到一半停在发送框里，亮得晃眼。
