# 第13章 投委会上绝地翻盘

玻璃门被推开的瞬间，临江市十一月微冷的风撞上出口的中央空调冷气，旋着绕上苏哲的领口。他捏着封皮磨得起毛的笔记本，目光扫过长条形投委会议桌——张广涛坐在主位斜对面，藏青色西装熨得没有一丝褶皱，指尖转着万宝龙钢笔，看见他进来，嘴角勾起一抹意味不明的笑。

王堔坐在苏哲同侧，指节叩了叩桌面，示意他落座。会议桌末端的林溪抱着笔记本电脑，指尖停在键帽上方，抬眼递来稍安勿躁的眼色。旁听席角落的陈曦埋着头，发丝垂下来遮了半张脸，没人读得透她的神情。

首位投委清了清嗓子，直接抛出议题：“今天临时开会，目的大家都清楚，讨论万客来项目负责人调整，张广涛，你之前递了申请，你先讲。”

张广涛立刻坐直脊背，推了推细框眼镜，语气是排练过百遍的恳切：“各位领导，我不是争位置，万客来是云帆科技今年重点押注的本地生活项目，绝不能砸在手里。苏哲接手上来，先搞什么内部梳理，把我带过来的老班子拆得七零八落，现在倒好，竞品抢跑提前公测，我们原定的三个核心流量广告位，昨天全被对方截了胡，明天就是万客来公测的日子，没流量项目直接胎死腹中，这个责任谁来担？”

他顿住话头，目光直直钉在苏哲脸上，语气裹着冰碴：“我不否认苏哲技术过硬，但做项目不是写代码，人脉统筹、资源整合样样跟不上，现在临机换将，还能把项目捞回来，再拖几天，砸进去的几千万预算，全得打了水漂。”

会议室瞬间落针可闻，所有视线齐齐钉在苏哲身上。苏哲指尖摩挲着笔记本磨起毛的封皮，心底半分波澜都无。放在前世，这个节点他早慌了，涨红了脸只会翻来覆去辩解自己没出错，最后被张广涛扣上能力不足的帽子，灰溜溜踢出项目组，背了全锅，房贷断供，三十岁那年纵身跳进了临江市的江里。这结局刻进骨头，是他重生回来就烙下的印记，接下来每一步走向，他比在场任何人都清楚。

可现在不一样。桌下的指尖轻轻点了点腕间的智能手环——那里面藏着来自十年后的GPT-8，从他重生回到这个关键节点那天起，这个LLM大模型就一直伴在他身侧，帮他拆解局势，生成全量应对方案。昨夜GPT-8还把今天所有可能的走向反复推演了三遍，张广涛可能抛出的每一张筹码都列得明明白白，连对方说话的语气停顿都标注得一清二楚。

苏哲抬眼，迎着张广涛的目光笑了笑，开口道：“你说的不对，第一，核心广告位没被截胡，第二，万客来明天公测，流量只会比原定计划多，不会少。”

张广涛愣了一秒，随即嗤笑出声，后背往椅背上一靠：“苏哲你这是病急乱投医？昨天我亲自跟三个老牌渠道的负责人确认过，他们的首页核心位早就签给竞品了，你现在上哪变出广告位来？靠嘴炮就能变出流量？”

“原来那三个位置确实给了竞品，”苏哲站起身走到会议室前端的投影幕前，接过林溪递来的U盘插进接口，“但我拿到了位置更好的新广告位。”

投影亮起，最上方是一份盖好骑缝章的流量合作合同，甲方云帆科技，乙方是临江市刚崛起的短视频本地生活运营机构。张广涛皱紧眉头，语气满是不屑：“什么野路子小公司？这种刚起盘的平台能有多少流量？你打算拿几千万预算打水漂？”

“2019年短视频日活已经破亿，本地生活板块刚放开流量扶持，现在进场，不仅不用抢核心位置，刊例价还比原来的传统渠道便宜两成，”苏哲点了点投影上色彩分明的用户聚类图，“这是GPT-8帮我做的用户画像分层，原来的渠道是广撒网通投，预估转化率不到两个点，我们按LLM分析出的标签做精准投放，转化率能到五个点，总触达用户比原定计划多三分之一。”

他顿了顿，切换到合作方的后台流量数据截图：“合同上周就签了，款项也付了，广告位已经预留，明天零点准时上线，数据随时可查，没有任何变数。”

王堔这时开口，把一叠复印好的合同和资质核查报告分到每位投委面前：“这个项目调整我提前知情，苏哲也跟我同步了所有细节，我已经派人核实过对方的资质和流量储备，确实没有问题，预算走的是项目备用金，没有超出原定范围。”

张广涛的脸色瞬间白了一度，指节攥得钢笔几乎变形。他明明算得准准的，陈曦说苏哲根本没察觉流量端的手脚，直到昨天还在改公测版本，怎么可能提前一周就签了新合同？他控制不住望向旁听席的陈曦，对方依旧埋着头，指尖划着手机屏幕，半个眼神都没分给他。这一切全落进苏哲眼底，前世张广涛就是靠这种信息差把他玩得团团转，这一世陈曦受够了张广涛的画饼反水，早把双重杀局的消息递到了他手里，他怎么可能不提前布局。

投委们翻完合同和预估数据，会议室里的风向瞬间转了，原本偏向张广涛的几个人也开始小声议论，说苏哲早就成竹在胸，临阵换将反而乱了军心。最终投票，张广涛的罢免申请以四票对两票被否决，苏哲继续留任万客来项目负责人，张广涛调出项目组，等候人力部门安排新岗位。

散会时，张广涛擦着苏哲身侧走过，压着声音咬牙：“你别得意，我还有后手。”说完头也不回地走了。

苏哲回到自己的工位，关上隔间门，打开电脑调出GPT-8的后台。刚才开会时，GPT-8就弹出了提示，说自动爬取竞品公测公开数据时，解锁了一项此前未激活的原生权限。他点开提示框，屏幕跳出一段他从未见过的代码界面，最下方一行字清晰显示：溯源完成，上传节点：临江市江滩，2029年6月17日。

那一天，是他前世投江的日子。

苏哲的指尖顿在触摸板上，原来GPT-8不是凭空掉下来的，是十年后临死的自己，拼着最后一口气传到这个时间节点，帮重生的自己破局？这个未知权限的边界，远比他预想的更深。

门被轻轻叩了两下，林溪端着一杯冰咖啡进来放在桌角：“内部邮件发了，团队那边都稳了，刚才渠道又确认了一遍，广告位没问题，明天公测准备得差不多了吧？”

苏哲关掉那个神秘的界面，抬头冲林溪笑了笑：“都准备好了，没问题。”

刚说完，手机震了一下，陈曦发来的短信，只有短短一句话：张广涛跟云帆的头号竞品签了对赌，万客来黄了他拿五百万，还有后手没出。

苏哲看着短信，窗外临江市的阳光落在屏幕上，亮得晃眼。重生这大半个月，他从那个只会被动迎合的程序员，变成如今敢跟张广涛正面硬刚的项目负责人，靠的不止是前世的记忆，还有GPT-8这个LLM的辅助，更重要的是，他终于敢主动攥住自己的命运。他把手机扣在桌面上，指尖落在键盘上，万客来公测的最后一行优化代码已经就绪，不管张广涛还有什么后手，他接着就是。
