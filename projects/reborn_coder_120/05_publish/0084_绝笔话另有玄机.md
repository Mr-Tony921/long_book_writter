# 第84章 绝笔话另有玄机

临江市的风已经带了凉意，新万客来租的写字楼办公室里，苏哲盯着电脑屏幕上云帆科技法务发来的最后通牒，指尖在桌沿敲得不急不缓。屏幕上的红字扎眼——三十分钟后，云帆合作的所有第三方支付机构将终止对新万客来的服务，所有商户提现通道全部冻结。

放在上辈子，这个时候他已经慌得手脚冰凉，连夜打了七八个电话给云帆的高层求情，放低身段愿意出让八成股权换一条活路，最后还是落得商户集体跑路，万客来彻底易主的下场。可现在，苏哲只是端起凉掉的美式抿了一口，重生回来这大半年，他太清楚王堔和云帆出牌的顺序，这招釜底抽薪，他三个月前就留了后手。

门被推开，林溪抱着一沓装订好的扫描件走进来，发梢还沾着外面的凉意：“都跑完了，自研LLM把所有手写内账都核验完了，你要的关联证据都标出来了。”她把U盘插在投影上，黑色的背景上慢慢跳出清晰的账目列表，这是之前从苏振海留下的老仓库夹层里翻出来的，是王堔二十年来记的逃税行贿原始内账，好多地方笔迹被故意涂掉，还有些只有王堔自己看得懂的暗语。换做以前，会计师要梳理完这些至少要三个月，还不一定能挖出隐形关联。但靠着适配了临江市本地商业数据的LLM，只用了十二个小时，所有涂改掉的金额、暗语对应的项目全部被推理还原，哪一年给原工商审批科科长送了一套市中心的商品房，哪一年把万客来的两千万营收转到云帆科技的空壳公司走账逃税，连王堔十年前帮云帆低价收购万客来股份收的那笔五百万好处费，都从几笔零散的小额报销里关联了出来，每一笔都对应着银行流水的时间和大致金额，刚好够当二审的核心铁证。

苏哲翻了一遍平板上的整理稿，指尖在那笔五百万好处费的条目上停了两秒，抬头对林溪说：“存公证处云端，备份加密。”

话音刚落，张广涛就推门撞了进来，眉头拧成了结，语气发急：“城西那十八家摇摆商户出问题了，王堔昨天夜里挨个上门找了，拿支付冻结的事吓他们，说只要我们通道一断，他们卖货的钱一分都提不出来，现在已经有三家放话要回去签原万客来的合同，剩下的也都在观望，今天约好了在便民市场等说法，咱们怎么办？”

苏哲笑了笑，起身拿起椅背上的外套：“怎么办？上门给他们看真东西。把上周给这些商户做的LLM增量引流数据导出来，带着平板过去。”

出门上车，陈曦握着方向盘透过后视镜看了眼后排的苏哲，开口说：“我昨天在云帆的内部群摸了消息，王堔这次不光要搅商户的局，还带了东西，说要当着所有人的面揭你‘造假’的底，我还听到他们提怀表，说王堔手里那块创始怀表，当年是你父亲苏振海亲手给他的。”

苏哲摸了摸内侧口袋里，那块用三块怀表零件拼出来的铁箱钥匙硌着掌心，他心里翻起浪。之前打开苏振海留下的铁箱，才知道自己的重生是父亲拿命换的，紧跟着就曝出王堔也带着前世记忆重生，那封绝笔信上写的“除了我，还有人能重启时针”，他一直以为这句话指的就是王堔也重生的事，可现在陈曦这么一说，倒像是还有别的隐情。四块创始怀表，三块在他这里，一块从苏振海死后就落在王堔手里，原来真的是苏振海亲手给的？为什么？

车很快开到城西便民市场，空地上已经摆好了折叠桌，十八个商户围站着，王堔正端着保温杯站在桌子另一边，看见苏哲过来，脸上立刻堆起熟络的笑：“苏哲，好久不见，我还以为你不敢来了。”

底下商户立刻窃窃私语起来，有人扯着嗓子问：“苏总，云帆真要冻我们提现？我这几天进货款还等着提呢，要是提不出来我可真扛不住。”

苏哲往前走了两步，摆手压下骚动，点开平板把LLM做的引流数据投在随身带的幕布上：“大家先看这个，上周我们给每家都做了定制引流，LLM根据你们店周边三公里的客群年龄、消费习惯推的精准优惠券，李记包子铺的李叔，你这十天早高峰客流涨了32%，昨天的流水你自己手机里能查，对不对？”

穿围裙的李叔愣了愣，摸出手机看了一眼，点头：“是真的，比之前多了小一百个客人。”

“巷口张姐的水果店，你那批进口晴王的券放出去，十天卖了八十七箱，比上个月整月卖的都多，对不对？”苏哲接着往下说，一连点了八家，每个报出来的增长数字都和商户自己的流水对得上，骚动慢慢停了。“支付的事大家放心，我三个月前就和两家持牌支付机构签了备用协议，现在通道已经开好了，十分钟就能完成切换，今天的提现照样到账，不会耽误大家一分钱生意。”

商户们顿时松了口气，不少人已经往苏哲这边靠了靠。王堔脸上的笑却一点没变，他慢悠悠放下保温杯，伸手从西装内袋里掏出那块铜制的创始怀表，放在掌心掂了掂，怀表的铜光在太阳下亮得晃眼。

“苏哲，说这些都没用，你夺回了商标，做好了引流，那你敢不敢告诉大家，你打开你爸留下的铁箱，到底漏了什么？”王堔往前走了一步，声音抬得足够所有商户听见，“你爸那句绝笔信里的话‘除了我，还有人能重启时针’，你不会真的以为，说的是我也带着前世记忆重生吧？”

苏哲的心猛地一沉，口袋里那把拼出来的怀表钥匙突然烫得惊人，他能清晰感觉到，钥匙上嵌着的时针，正不受控制地，慢慢往回转了半格。
