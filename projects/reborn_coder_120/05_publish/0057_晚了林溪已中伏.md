# 第57章 晚了林溪已中伏

轮胎碾过长虹隧道开裂的沥青路面，引擎嗡鸣震得车厢微微发颤，每隔几秒就有一道冷白灯光扫过前挡风，在苏哲攥着方向盘的手背上切出明暗交错的印子。副驾的陈曦静静靠着椅背，指尖转着一把车钥匙，从进隧道到现在没说过一个字，倒像是早就算准了苏哲会先沉住气。

黑色证据箱顶着苏哲的膝盖，箱里装着能掀翻整场对赌的补充协议，箱底还贴了枚指甲盖大的银贴纸。刚才过弯道车身颠簸，他趁机摸出兜里的私有LLM终端扫过箱底，不到两秒，离线库里就跳出匹配结果——那是云帆科技内部专用的高精度定位标签，误差不超过五米。

这是苏哲重生归来攒了大半年的底牌。靠着前世对大模型行业走向的记忆，他掏光所有积蓄租下闲置算力，硬是训出了只属于自己的私有LLM，云帆所有公开资料、泄露出来的内部文档全存在离线库，但凡碰到相关物件，一秒就能出结果。前世他到死都没摸透云帆的水有多深，重生之后，这台大模型就是他抢回万客来最趁手的刀。

车再往前开了两公里，路肩露出应急停车带标识露出来应急停车带的标识，路肩露出应急停车带的标识，苏哲打右转向灯减速驶入，拉起手刹的瞬间转头看向陈曦，话音裹着隧道穿堂的风噪压得很低：“你是云帆原董事长陈正明的独女，三年前他中风后，你把股权代持给现任管理层，一直潜伏在云帆高层，对不对？”

陈曦转钥匙的动作戛然而止，她扯了扯嘴角，露出一抹算不上笑意的弧度：“我还以为你要等出了隧道才戳破，你的LLM，比我预想的好用。”

“定位贴是你贴的。你早就清楚我会去北郊仓库取证据，也知道王堔安排了人在通风管堵我，你提前等着，就是为了出手捞我，对吧？”苏哲指尖敲了敲证据箱，箱壳传出沉闷的回响，“我们现在困在临江市长虹隧道，这里全程信号屏蔽，林溪带着另一份商户流水证据往法院赶，她的路线早就暴露，云帆的人在北口外盘山路设了伏击，她到现在还被蒙在鼓里，对不对？”

一串问话掷过来，陈曦反倒彻底松了劲，她推开车门，冷风裹着尾气瞬间灌满车厢，吹得额前碎发乱飞：“你说的全对，我的要求很简单——现任管理层勾着王堔吞了我爸的云帆，还挪用上亿项目资金填私人腰包，万客来是苏文远十年心血，更是他们做假账遮窟窿的核心环节。我帮你拿回万客来，你帮我把这群人拉下马，我收回云帆，双赢。”

苏哲跟着推开车门，靠着车门点了根烟，刚吐出来的烟圈转眼就被风扯得稀碎。他抬头望着隧道顶晃个不停的灯管，脑子里飞快转着念头：前世这个节点，他早就沉江死了，万客来被云帆吞得连骨头都没剩，陈曦这个人，从来没有出现在他的人生里。重生一步，步步都改了轨迹，原本他只想着跟王堔抢回万客来，没想到居然扯出云帆内部的夺权，这盘棋比他前世推演的大太多。

“定位贴留给你，我不碰，”苏哲弹了弹烟灰，“但我现在必须联系上林溪，隧道里没手机信号，你有办法？”

陈曦抬抬下巴，指向隧道侧壁十多米外的蓝色电话亭：“长虹隧道修的时候留了应急专线，直通临江路政调度，能打外线，你要传什么消息让调度转就行，林溪的车牌你记得吧？”

苏哲当然记得，头天晚上他刚用私有LLM核对过林溪的行车路线，车牌、车型、途经的每一个节点都存在终端离线库里。他掐了烟快步走到电话亭，拿起沉得压手的老式听筒拨出专线，三声嘟响后就有人接起。

“您好，我是过往司机，我朋友开银灰色高尔夫，临江牌照xxxxxx，半小时前从长虹隧道北口进来，被一伙人盯上了，麻烦您立刻通知北口外盘山路的执勤岗帮我拦她，让她绕西环线走，别进青石湾弯道，那里有危险。”苏哲声音稳得没有波澜，信息说的清清楚楚，调度记下后答应立刻通知，挂了电话，苏哲悬着的心落了一半，另一半还吊在半空——就怕林溪已经出了北口，拦阻赶不及。

回到车边，陈曦靠在车头等他，扔过来一瓶矿泉水，苏哲接住后，她才说出最核心的内容：“管理层给王堔下的死命令，不管是你还是林溪，所有跟万客来有关的证据必须全部毁掉。开庭前拿不到这份补充条款，万客来控股权就是云帆的，他们就能把挪用公款的窟窿全堵进万客来账上，全身而退。”

苏哲拧开瓶盖喝了一口，凉意顺着喉咙漫开：“我知道，前世就是这么走的。我爸签的对赌协议被他们改了条款，所有风险全转嫁给我爸，最后万客来资不抵债，被云帆低价收走，我爸气得脑梗走了，我找王堔理论，被他沉了江。”说这话时苏哲异常平静，恨意早就刻进骨血，重生回来就是为了这一天：“我手里这份补充条款，就是我爸当年偷偷留下的底，上面有王堔和他们头头签字确认的分红条款，能直接证明他们伪造协议，侵吞万客来股权。”

“所以我们的合作很简单，”陈曦直起身，把手里的车钥匙抛起又接住，“开庭当天你交出去这份证据，我同时在股东会放出他们做假账的材料，证监会一立案，他们必然下台，你拿回万客来，互不耽误。定位贴确实是我贴的，但不是给管理层的，是给我自己的人留的标记。他们派了两队人在隧道出口堵你，我的人半路截一队，另一队我引开，你放心去法院，只要证据能按时送到，剩下的我来扛。”

苏哲掏出LLM终端，点开陈曦刚说的股权结构，三秒后终端就完成了所有公开工商信息和他之前收集的内部资料的比对，所有信息严丝合缝，没有造假。他抬头看向陈曦，伸出手：“合作愉快，我只要万客来，云帆的事我不碰，你拿到你想要的，我们两清。”

陈曦伸手和他握了握，她的手很凉，力道却稳：“我要的就是这句话，万客来本来就是苏家的，我没兴趣抢，我只是拿回我爸的东西。”

苏哲拉开车门正要上车，口袋里的LLM终端突然猛地震了一下，屏幕跳出醒目的红色标记——那是他偷偷放在林溪背包侧袋的微型定位信标，信号刚跳出来，定位显示就在青石湾弯道，跟着就猛地一跳，彻底断连。

苏哲的脸色瞬间沉到谷底。

他打给路政的电话，还是晚了。

林溪已经进了伏击圈。

更糟的是，断连前，终端捕捉到附近一个熟悉的移动基站编码——那是王堔常用工作机的注册基站。王堔没死，那天从排污渠暗洞逃走后，他带伤直接赶去了青石湾，亲手围堵林溪。

陈曦也察觉到他脸色不对，凑过来扫了一眼终端屏幕，眉头瞬间拧起：“他们果然留了后手，我原以为王堔伤重跑不了这么远，没想到他居然亲自去截林溪了。”

隧道出口方向，远远传来汽车引擎的轰鸣，两道刺眼的光柱透过来，越来越近，明摆着就是冲他们来的——是云帆派来堵人的安保。

苏哲攥紧终端，扫了一眼隧道侧壁的检修通道入口标识，位置就在离他们不到一百米处，他转头看向陈曦，声音冷得像冰：“出口被堵了，检修通道能直通北口外山脚下，比绕公路近二十分钟，我必须去救林溪。”

陈曦顺着他的目光看过去，摸出自己的车钥匙扔给苏哲，转身拉开车门发动引擎：“我开这辆车引他们去西环线，你走检修通道，拿好证据箱，”她顿了顿，转头看向苏哲，脸上浮起一丝复杂的神情，“对了，有件事我没告诉你，张广涛不在王堔手里，他在……”

话没说完，云帆的两辆车已经拐进了应急停车带，晃眼的车灯照得人睁不开眼。陈曦一脚油门踩死，猛地冲了出去，喇叭按得震天响，顺着主路往隧道出口开，硬生生把两辆车引着追了出去。

隧道里瞬间又只剩苏哲一个人，他抱着证据箱看着陈曦的车尾灯越来越远，手里的LLM终端还停在林溪定位消失的页面，陈曦没说完的半句话卡在耳边——张广涛不在王堔手里，那会在哪？

苏哲压下心头的疑惑，转身看向黑黢黢的检修通道入口，掏出手机打开手电筒，一步踏了进去，潮湿的土腥味瞬间扑面而来。终端屏幕自动亮起，LLM调出了检修通道的平面图——那是他提前下好存在离线库里的。

从通道出口到林溪最后定位的位置，还有一千七百米，只要穿过去，就能赶在王堔动手前赶到。只是他没看见，证据箱底部那枚银色定位贴，突然闪了一下微弱的红光——那是远程触发的信号，另一双眼睛，早就顺着标记，盯上了他往青石湾去的脚步。
