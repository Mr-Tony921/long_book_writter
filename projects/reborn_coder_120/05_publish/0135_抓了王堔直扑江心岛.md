# 第135章 抓了王堔直扑江心岛

江风卷着码头的柴油味往领子里钻，王堔攥着拉杆箱把手，指节泛出青白色，箱盖缝里露着银灰色硬盘盒的边。他脚边扔了半根抽完的烟，烟蒂还在湿沙子里冒着细烟，远处的江面上雾蒙蒙的，根本看不到接应船的影子。

苏哲靠在三米外的卸油桩上，手机屏幕亮着淡蓝色的光，那是他重生后靠着上辈子的记忆攒出来的LLM分析工具，刚跑完王堔的出逃路径推演。2019年的临江，还没多少人玩得转大模型辅助办案，苏哲这口手艺，帮他省了至少八个小时的堵截时间，也掐断了王堔最后一条逃生路。

“苏哲，万事留一线，”王堔的嗓子哑得像砂纸磨过，“你不就是想要云帆那点核心语料？蓝星开的价是三百万，我分你两百万，硬盘给你，我换个身份远走高飞，谁也不耽误谁，不好吗？你刚在云帆站稳脚，犯不着为了这点事把人得罪死。”

苏哲挑了挑眉，没接他的条件，只反问：“接应你的船呢？说好十分钟到，这都半个钟头了，怎么还没见影子？”

王堔脸色瞬间变了，手下意识摸向腰后藏的水果刀：“你动了什么手脚？”

“半小时前LLM就截了你的对讲频道，信号我给你中转到别的频段了，”苏哲晃了晃手机，屏幕上跳动着几个红色的坐标点，“我还给你那船发了十公里外的浅滩对接坐标，现在正是退潮，那片滩涂插满了旧船桩，他们现在正忙着掏船底的淤泥，估计天黑之前都过不来。”

王堔的脸瞬间灰得像码头泡了水的木头，他往前跨了一步想要拼命，又被旁边阴影里冲出来的刑警狠狠踹在膝盖弯，噗通一声跪在了碎石子码头上。尖利的碎石蹭破了他的牛仔裤，渗出来的血印子很快晕开在深色布料上，他撑着地面抬头骂：“苏哲你个小王八蛋！我在云帆熬了十五年，凭什么你一个刚毕业没几年的毛头小子就能骑在我头上？不就是你运气好抢到了大模型项目？”

“运气好？”苏哲走过去，蹲下来看着他的眼睛，语气冷得像江面上的风，“上辈子你和蓝星勾结转走核心语料，我爸急着找你对质，淋了一夜秋雨突发脑溢血，走的时候才五十二，你跟我讲运气？那时候你拿了钱在海南度假，怎么没想着给我留半条路？”

这话戳得王堔愣在原地，他张了张嘴，半天没说出一句整话，只盯着苏哲的眼睛，那眼神里的恨意太沉太浓，根本不像二十多岁的年轻人该有的，倒像是攒了一辈子的仇。“你……你胡说八道什么疯话？”

“我疯不疯，你进去慢慢想，”苏哲抬了抬下巴，旁边的刑警上前，冰凉的手铐咔哒一声扣在王堔手腕上，那金属的凉意顺着骨头渗进去，王堔瞬间垮了肩，整个人软了下来。

刑警按着王堔搜身，从他的内兜摸出一个塑封的小卡片，擦干净表面的烟灰递过来给苏哲。那是一张早就过期的临江江心岛生态农庄的游船招待卡，正面印着歪歪扭扭的桃花广告，背面用铅笔淡写了一行字：三号木屋，凌晨接头，末尾还留了半个淡粉色的指纹印，应该是捏卡片的时候沾了茶水印上去的。

苏哲捏着卡片转了一圈，王堔被架起来准备往警车走的时候，突然疯笑起来，声音劈得厉害：“你们抓我有什么用？渔翁早就等着了！整个云帆都在人家口袋里，你苏哲就算抓了我抓了陈曦抓了张广涛，又能怎么样？渔翁照样把东西带走，你还是什么都捞不着！”

“渔翁？”苏哲捏着卡片的手指顿了顿，“陈曦只说蓝星有个幕后联系人，原来代号叫渔翁？”

王堔闭了嘴梗着脖子，再也不肯多说一个字，被刑警押着往外走，脚步重得像踩在棉花上，整个人都没了精气神。

刚押走，苏哲的手机就响了，屏幕上跳着林溪的名字，他接起来，听筒里传来林溪带点喘气的声音，背景是高速开过的呼呼风噪：“苏哲，我到陈曦说的地方了，就是城郊那个废弃IDC机房，保险柜没错，我打开了。对了，那锁真够邪门的，自带延时触发的自毁装置，我按照你之前说的，用你给的那个LLM解码小程序扫了锁芯的纹路，三分钟就解开了，没触发爆炸，东西完好无损。九十G完整的大模型微调语料备份，五个硬盘全在，都没拆封，也没查到被拷贝过的痕迹。我已经做了三个异地镜像，一个放我随身加密硬盘，一个传了云帆的离线加密服务器，一个寄存在我师姐的保险柜，绝对安全，你放心。”

苏哲悬了快一个月的心，这时候终于踏踏实实落了地。重生这一趟，他拼了这么久，最核心的目标就是保住这份语料——这是他做国产大模型的根，是上辈子没保住丢了的东西，现在完完整整拿回来了。“路上开慢一点，回市区之后先找地方歇着，别去公司，后续我给你消息。”

挂了电话，苏哲把那张游船招待卡翻过来，调出手机里的LLM分析工具，把指纹印的高清照片传了上去。不到一分钟，比对结果就跳了出来：和已经落网的张广涛、陈曦的指纹都不匹配，属于未知第三人。而且卡片表面除了指纹，还有淡淡的海水盐渍残留，说明这张卡最近一直带在人身上，飘过江风。

负责带队的刑警走过来问苏哲，人已经拿到了，要不要一起回市局做笔录走流程？苏哲摇摇头，把卡片揣进牛仔裤兜，指着码头外自己的车说：“你们先送王堔回去，流程按规定走就行，我这边还有点事要处理，后续补笔录也一样。”

刑警知道苏哲是这次办案的线人，线索全是他摸出来的，也不多问，转身上车押着王堔走了。码头上很快安静下来，只剩下江风撞着卸油桩的声音，还有远处江轮的鸣笛声。

苏哲靠在自己车的引擎盖上，翻出之前审讯陈曦的记录，陈曦只说她和王堔都是给上游联系人干活，从来没见过对方，只知道对方每次接头都选在江心岛，这次王堔出逃的路线，也是对方安排的，事成之后就在江心岛交接语料。原来张广涛、陈曦、王堔，都只是摆在明面上的棋子，真正的渔翁，一直躲在江心岛等着收成果。

他拧开矿泉水瓶喝了一口，凉水流过喉咙，把刚才攒着的火气压了下去。现在王堔已经拘押，张广涛陈曦都落网，核心语料也找回来了，蓝星这次侵吞云帆科技核心资产的布局，已经破了大半，可偏偏这个藏在背后的渔翁，才是最关键的那个漏网之鱼。

苏哲把矿泉水瓶拧紧，拉开车门坐进去，发动车子，方向盘一转，就朝着通往江心岛的过江大桥开过去。导航屏幕上，江心岛的光点越来越近，他摸了摸兜里那张带着指纹的游船卡，手机上的LLM工具已经提前加载好了江心岛最新的卫星图，三号木屋的位置，清清楚楚标在了岛西北角的枫树林里。

绕了这么大一圈，真正的对手才刚露了头。这一趟江心岛，正好会会那个躲在幕后的渔翁。
