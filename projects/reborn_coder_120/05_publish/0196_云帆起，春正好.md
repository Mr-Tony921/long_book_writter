# 第196章 云帆起，春正好

江风卷着二月的桐花落在苏哲臂弯，他低头碾了碾鞋跟沾的淡紫色落蕊，身边林溪递过来一瓶温矿泉水，指尖不经意蹭过他的手背，带着露天停车场晒了半天的暖意。今天是苏振江引渡回国后的终审宣判日，临江市中级人民法院门口的百年梧桐刚抽新叶，风一过就落一地碎影，挤在门口等结果的媒体记者早早就散了，只剩苏哲他们三个慢腾腾往外走。

苏振海走在最前面，藏青色夹克的衣角被风掀起来，他背挺得很直，只有鬓角的霜白看得出十年牢狱磨出来的痕迹。上周法院刚下达了再审判决书，撤销了当年对他挪用公款的指控，改判无罪，压在苏家头顶十年的乌云，终于彻底散了，那桩当年轰动临江市创投圈的旧案，彻彻底底盖了棺。

苏哲追上去递烟，苏振海摆了摆手，笑着说早就戒了，肺不好，留着命看你把船开起来呢。法院门口的台阶上，苏振江的律师早垂着脑袋走了，终审判决书上写得清楚，数罪并罚，无期徒刑，没收全部非法所得，当年苏振江卷去海外的三亿两千万资产，通过跨境追逃追赃，追回来两亿七千万，一分不少都打回了新云帆的对公账户。刚好填上了首款产品研发的最后一块缺口，也补上了老云帆十年前破掉的窟窿，跨境的收尾，总算是漂漂亮亮落了地。

“张广涛和王堔那边呢？”林溪走在苏哲身边，轻声问。苏哲踢开脚边一颗小石子，声音很平：“张广涛去年就判了，职务侵占加商业诽谤，十二年，上个月已经移去监狱服刑了。王堔受贿罪滥用职权，八年，没收所有受贿所得，早就双开了，没什么波澜。”陈曦当年是被苏振江胁迫做了假证，后来主动交出核心技术备份，配合警方锁定了苏振江的海外落脚点，立功免予刑事处罚，早结清了所有旧账，三个月前就背着包去西北了。这些话苏哲没说，林溪也都知道，只是问一句，把所有该摆出来的事都落个实，给这段牵扯了十年的恩怨画个句号。

车开出去二十分钟，就到了苏振海江边的院子。这是苏振海用追回来的部分旧资产买的，临着江，院子不大，墙角移栽了一棵桐树苗，是从当年老云帆旧厂房的院子里移过来的，现在抽了半人高的新枝。苏振海推开门，把手里的鱼篓往墙角一放，笑着说：“我昨天刚钓了两斤鲫鱼，一会儿炖汤，你们尝尝江里的鲜。”

苏振海现在的日子过得闲，国家赔偿下来的八十万，他转头就捐给了临江市大学的AI专业奖学金，说当年就是靠奖学金才读得起书，现在给后辈搭个桥。新云帆的董事会请他当名誉董事长，他推了，说自己年纪大了，跟不上现在的技术迭代，只想安安稳稳过日子，偶尔苏哲研发遇到卡壳的基础问题，他过来把把关，平时就是整理整理老云帆当年的技术笔记，早上去钓鱼，下午在院子里晒太阳，归置归置花。十年的冤屈磨没了他的棱角，却没磨掉他对技术的热乎劲，苏哲知道，父亲这是真的回家了，颠沛了十年，终于归位了。

吃完饭坐了一会儿，苏哲和林溪开车回新云帆的办公楼。园区就在临江高新区，去年才交的楼，前台挂着烫金的牌子“新云帆科技”，字还是苏振海写的。研发部在十二楼，推开门的时候，整个部门都在忙着做最后的内测调试，首款轻量化通用LLM“轻帆”，下周三就要正式上线了，悬在苏哲心头大半年的石头，终于要落地。

研发主管拿着平板跑过来，额头上满是汗，语气却亮得很：“苏哥，刚才后台统计，预约开放接口的中小开发者已经突破三千家了，还有三个高校的实验室发了邮件，要跟我们合作做垂直领域的模型调优，上周工信部的数字经济创新专项也批下来了，一千万扶持资金，不用稀释股权，下周就到账。”苏哲接过平板翻了翻内测的用户反馈，大部分都是说轻帆的参数小，响应快，对中小团队太友好了，比买国外大模型的接口便宜一半还多。他笑了笑，拍了拍研发主管的肩膀：“注意稳，上线前把所有bug再跑三遍，不急着抢流量，我们先把口碑做起来。”

十年前老云帆死的时候，苏哲就暗下决心，要做一款普通人用得起、用得顺手的国产大模型，不炒概念，不割韭菜，只给真正需要技术的人搭台子。现在这个愿望，终于要落地了，他的事业，真的站稳了。

林溪靠在研发部的玻璃门上，看着苏哲跟团队讨论参数调整的样子，掏出手机翻了翻，突然笑出了声。她走过来碰了碰苏哲的胳膊，把手机递给他看：“你看，陈曦刚发的朋友圈。”屏幕上是西北戈壁上的希望小学，红墙国旗，一群晒得黑红的小孩子围着平板挤在一起笑，镜头边角能看到陈曦扎着马尾的半个身子，配文是“第一个AI识字班开课啦，三个月已经教会二十个孩子认一千个字了，等葡萄熟了，欢迎来玩。”

苏哲笑了，陈曦当年一直活在苏振江的阴影里，觉得自己对不起老云帆对不起苏振海，现在在西部找到了自己的价值，比困在临江市的写字楼里内耗强多了，这才是她最好的归宿。

傍晚的时候，两个人又晃回了江堤，坐在防波堤的石阶上看日落。夕阳把江面染成一片橘红，风裹着桐花的香吹过来，林溪突然从包里摸出一个用蓝布包着的小东西，递到苏哲面前。

苏哲愣了愣，打开蓝布，是个小小的木质帆船吊坠，木纹磨得发亮，船帆上还刻着半个歪歪扭扭的“哲”字。他一下子就想起来了，这是他大二那年，在金工实习课上自己磨的，那时候他刚喜欢上林溪，想送给她又怕被拒绝，后来老云帆破产，他搬东西的时候弄丢了，没想到……

“我去年跟陈曦去旧仓库找技术备份的时候捡到的。”林溪指尖摸着那小帆船，眼睛弯成了月牙，夕阳落在她的眼睛里，亮得像星星，“你当年磨这个，是不是有话没说？”

苏哲攥着那小小的吊坠，木头还带着林溪口袋里的温度，十年攒着的情绪一下子涌上来，堵在喉咙口。他看着林溪的眼睛，声音有点发哑，却很稳：“那时候我什么都没有，老云帆快没了，我连一份稳定的未来都给不了你，不敢说。现在我有新云帆，有攒够了的勇气，我想说，我喜欢你，从十七岁在实验室门口你给我递了一块橡皮开始，我就喜欢你了。”

林溪笑了，眼睛里浸出点水光，她接过硬币，抬手挂在自己脖子上，吊坠落在领口，刚好贴在心口。她往苏哲身边靠了靠，肩膀贴着肩膀，江风把两个人的头发吹得缠在一起。

江面上远远开过来一艘新货轮，白漆的船舷上印着小小的云帆标志，鸣着汽笛往入海口去。林溪指着那艘船，半天，突然轻轻戳了戳苏哲的胸口，声音软乎乎的带着笑：“对了，跟你说个事，下周陈曦那边缺人做教育模型的适配，我申请了过去支援，要去两个月。哦对了，等我回来，轻帆也上线了，我有件很重要的事要跟你说，你可得提前做好心理准备。”

苏哲捏了捏她垂在身侧的手腕，低头看着她发顶的旋，笑着问是什么事，不能现在说吗？

林溪摇了摇头，把脸埋在他的肩膀上，江风卷着浪拍着堤岸，远处的汽笛声慢慢淡下去，夕阳把两个人的影子拉得很长很长。

“不急呀，”她轻声说，“我们还有一整个春天，等回来再说。”
