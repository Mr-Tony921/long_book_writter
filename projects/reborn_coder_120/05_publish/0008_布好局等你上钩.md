# 第8章 布好局等你上钩

江风裹着江岸栀子花香钻进衣领时，苏哲扫过腕间运动手表——六点五十八分，比约定时间早两分钟。他把搭在臂弯的休闲外套往上拉了拉，遮住领口别着的微型录音笔，推开云上咖啡厅的玻璃门。

临江市入夏的晚风裹着潮气，被店内冷空调一卷，混着咖啡香漫开。苏哲目光扫过全场，一眼就落在角落落地窗那桌——张广涛穿熨得笔挺的浅蓝衬衫，陈曦坐在对面，指尖叩着冰美式杯壁，嘴角勾着只有算计人才会露出的笑。

苏哲没径直走过去，转身拐上二楼露台，挑了斜对面的位置——既能看清两人唇形，又能清清楚楚接住对话。点完黑咖啡，他指尖悄悄按开录音笔开关，口袋里的手机恰好震了震，亮起来的屏幕跳着林溪的消息：我在停车场车里待命，王堔那边已经打点好，拿到实锤直接走开除流程。

苏哲回了个“收到”，指尖点开藏在工具文件夹深处的GPT-8后台，淡蓝色小字逐行跳出：剩余配额0.87%，预计可用时长四小时二十八分，请确认使用需求。

重生归来这一个多月，靠着这台来自未来的大模型，苏哲一次次拆穿张广涛的小动作，从刚入职的新人，硬生生把本该属于自己的万客来项目主导权攥在了手里。前世就是在这里，张广涛联手游走在他身边的前女友陈曦，把他坑得身败名裂：万客来上线前核心架构泄露，上线当日支付系统直接崩盘，所有黑锅全扣在他头上，被开除后房贷断供，三十岁生日那天，他纵身跳进了临江。重来这一遭，苏哲从没想过退一步。

骨传导耳机里混着咖啡厅的背景音，张广涛压得极低的话音飘上来：“东西拿到了？”

陈曦的话音裹着得意飘出来：“昨天趁他去投委会开会，我翻工位拿走的，他根本没察觉。说起来苏哲这阵子防我跟防贼似的，我翻了半天才找出那个U盘，亏得他还锁抽屉里，原来连加密都没做。”

苏哲端着黑咖啡抿了一口，身形没动分毫。那个U盘本就是他故意留在抽屉里的，里面存的是半年前他做的第一版万客来架构，底下埋了三个逻辑陷阱，每一行代码都带隐形水印，只要对方敢用，一查一个准。陈曦只当偷到了宝贝，哪里知道自己从踏入局的第一步，就掉进了苏哲挖好的坑。

“那就好办了，”张广涛的话音松快下来，“我跟那边早就谈妥了，下周离职就过去，直接做技术总监，总比在云帆受苏哲那小子的气强。等我把架构交过去，他们半个月就能改完上线，抢在万客来前头推出去，到时候投资人只会认苏哲抄袭，项目一黄，投委会自然把他踢走，咱们俩稳赚不赔。”

“你别忘答应我的东西，”陈曦哼了一声，“我帮你做了这么多，那套临江市商圈的用户资料得给我，我升运营主管全靠这个。还有，支付授权我改完了，走的是我手里的对接流程，下周三万客来内部测试就生效，到时候支付接口直接跳去对方收款码，苏哲连哭都找不到地方。”

苏哲指尖微微一顿——果然狠。这两人攒了半个月的阴招，一边偷架构给竞品抢发，一边在内部动了支付接口的手脚，只要上线测试必然炸锅，不管结果如何，他这个项目主导都脱不了干系。

苏哲没急着下楼，指尖在GPT-8输入框里飞快敲击，把刚听到的所有信息拆分录入，请大模型辅助梳理出完整证据链：张广涛串通竞品的时间线、陈曦修改支付授权的操作路径、旧架构的水印溯源说明，全部整理成加密文档。靠着重生带来的先知，他清清楚楚记得前世万客来上线后才爆发的每一个隐藏bug，借着LLM的算力，他把所有bug的修复方案、规避路径整理成标准化开发手册，连未来对接商家的接口规范，都按照未来五年的通用标准做了预留，比当下市面所有同类文档都严谨规范。

紧接着，他把刻在脑子里一辈子的万客来全量架构、前世项目成长途中踩过的所有坑、未来三年本地生活服务平台的迭代方向，全部口述录入，让GPT-8整理成结构化开发文档，分片加密后分别存进本地移动硬盘，还有一份同步到云帆科技内部加密服务器，权限仅对苏哲和王堔开放。

GPT-8的后台配额一点点往下掉：0.8%…0.3%…0.1%…最终停在0.01%时，淡蓝色弹窗跳了出来。不是苏哲预想的绑定解除提示，一串乱码闪过之后，跳出一行模糊的字：测试配额耗尽，原生适配启动，未知权限激活中。

苏哲刚蹙起眉想点进去查看，弹窗已经消失，后台退回初始界面，剩余配额停在原地，也没有弹出绑定解除的提示。这个意外变数来得猝不及防，他先锁了屏把手机揣回口袋——反正核心内容都已经落地存好，不管是什么变数，他都接得住。

楼下张广涛已经起身准备离开，苏哲理了理外套，捏着录音笔往楼下走，刚到楼梯口就撞了个正着。

张广涛脸上的慌色刚冒出来就压下去，很快堆起那副惯有的虚伪笑：“哟，苏哲，这么巧？也来喝咖啡？”

陈曦跟在他身后，脸色瞬间褪得雪白，下意识把手里的U盘往包里塞：“你…你怎么会在这里？”

“我不来，怎么能听到这么精彩的对话？”苏哲靠在楼梯扶手上，拿出录音笔按下播放键，张广涛说要带架构投竞品的话音清晰传出来，整个咖啡厅的目光都转了过来，“张广涛，陈曦，云帆科技发你们薪水，就是让你们联手坑公司项目的？”

张广涛脸上的笑瞬间挂不住，声音陡然冷下来：“苏哲你别血口喷人！我不过跟陈曦吃个饭，你偷偷录音算什么东西？”

“是不是血口喷人，咱们找王堔说清楚，”苏哲拿出手机，翻出提前截好的支付授权修改记录，“陈曦，你三天前凌晨两点登录运营后台，改了万客来的支付授权对接域名，IP地址就是你租住小区的，需要我把运营商记录调出来对证吗？还有你手里那个U盘，里面的架构每一行代码都带我的隐形水印，张广涛你要是敢拿给竞品，信不信我直接把你告上法院，让你吃几年牢饭？”

张广涛的脸瞬间铁青，他怎么也没想到苏哲早就把一切算透了，那个U盘根本就是故意留给他的。“你早就设好局等我钻？”

“不然呢？等着你们把我赶下来，再像前世那样把我逼死？”这句话说得极轻，只有三个人能听见，也只有苏哲自己知道，这话是说给重生的自己听的。前世他懦弱讨好，换不来半分留情，这一世他主动布下局，从来没打算给任何人留后路。

话音刚落，张广涛的手机响了，是HR总监打来的，说王堔已经通知投委会，同意了他的离职申请，明天就可以办手续，工资结算到当日，竞业协议已经发去他的邮箱，要是敢去竞品做万客来同类项目，公司直接按协议索赔。

张广涛挂了电话，死死盯着苏哲，咬着牙撂下话：“算你狠。但我告诉你苏哲，我早就把核心内容同步给竞品了，他们就算不用那版架构，也能凭着我给的信息提前做出来，你们万客来上线也抢不到流量，你照样守不住这个项目！”

说完，他猛地推开咖啡厅门扬长而去，陈曦愣在原地，对上苏哲冷得像冰的眼神，咬咬牙也跟着跑了。

苏哲站在咖啡厅门口，江风卷着栀子花香刮过来，吹得耳朵发疼。林溪开车过来停在他身侧，摇下车窗问：“都搞定了？王堔让你明天去他办公室，投委会要确认万客来的后续开发计划。”

苏哲拉开车门坐进去，把录音笔递给林溪，摸出口袋里的手机，看着后台那行没消失的模糊提示，低声说：“只搞定了一半。张广涛已经给竞品递了消息，接下来就看谁动作快了。”

车顺着江滩公路驶出去，路灯光斑一片片扫过苏哲的脸颊，他看着窗外流动的临江市夜景，心里透亮：GPT-8的测试配额用完了，未知的变数还在前面，张广涛的反扑也不会停，但这一次，他已经把所有底牌牢牢攥在自己手里，就算没有LLM辅助，他也能赢。
