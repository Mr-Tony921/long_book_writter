# 第47章 刚得模型就中套

苏哲指尖叩着磨砂玻璃会议桌，指腹蹭过桌面残留的烫金万客来logo。桌对面的陈曦手腕被绑在桌侧椅腿上，唇色泛白却仍勾着冷笑，半个字多余的都不肯说，只反复咬着一句话：核心基线模型早半个月就转出临江市，明天境外资本就能完成交割，苏哲拿到的，不过是空壳万客来。

林溪抬手扣住陈曦翘起来的下巴，不轻不重的力道压得她偏过脸，转身冲苏哲晃了晃手机：“云帆往来邮件查到上周就断了，之后的记录全删干净，保险柜里只摸出一张境外匿名银行卡，查不到开户人信息。”

苏哲颔首，示意林溪先把陈曦押去隔壁休息室锁好，等明天经侦过来交接。门合上的刹那，藏在走廊阴影里的张广涛夹着笔记本走了进来，后颈的汗已经洇透了浅蓝POLO衫的领边，指节绷得泛白。

“我知道你不信我。”张广涛先开了口，声音哑得发涩，“当年你爸带我出来闯，我欠他一份情，王堔那杂种拿我儿子的手术费逼我，我没办法才跟着他走歪路，但我从来没真想把万客来卖得一干二净。”

苏哲靠回椅背上，视线扫过张广涛捏得发皱的笔记本页边——重生回来这大半个月，他靠着前世记忆，早就用提前攒出算力训练好的私有LLM，跑完了万客来核心团队所有人三年来的资金流水和通讯记录。张广涛账户里那笔三百万的不明进项，早就在LLM的风险预警里标了红，他没点破，等的就是张广涛自己开口。

苏哲推了一杯冰矿泉水到他面前，开口问：“基线模型转移那天，你在哪？”瓶身凝的水珠顺着瓶壁滑下来，打湿了桌角叠着的文件。

张广涛指尖扣着矿泉水瓶身，指腹蹭得瓶身发响：“那天王堔逼我把模型打包发去境外中转服务器，我趁他们盯着陈曦做交割准备，偷偷拔了原模型的硬盘，换了一包训练完的废弃数据上去。原硬盘我藏在临江市郊外旧IDC机房第三排机柜的挡板后面，那是万客来最早的机房，后来云帆接盘的时候没动这块资产，我一直按月交租金，没人会去查。”

苏哲没接话，点开随身带的笔记本电脑，屏幕跳出他自己训练的私有LLM交互界面，把张广涛报出的机柜编号和地址输进去。LLM自动调取万客来早年的机房资产表匹配，三秒就跳出匹配成功的绿色对勾，连当年张广涛登记的租户信息都完整拉了出来，分毫不差。

苏哲抬眼看向他，开口问：“留这一手，你图什么？”

“我图心安，也图个退路。”张广涛叹出口气，蹲在会议室墙角点了根烟，淡蓝色烟圈裹着他的声音飘出来：“王堔是什么人我清楚，当年帮他坑你爸，我就知道早晚是他过河拆桥的棋子。境外资本要的就是模型，拿到手第一个就弄死我灭口，我留着模型，就是留着跟你们谈的资本。苏哲，我把模型交出来，只要你帮我结清我儿子的手术费，再给我们父子俩新的身份，我帮你搞垮王堔和云帆。”

话音刚落，苏哲放在桌上的手机就震得桌面轻轻响，来电显示正是王堔。苏哲接起电话开了免提，王堔漫不经心的笑意顺着听筒飘出来，满是挑衅：“苏哲，年轻人就是气盛，你真以为拿个控股权就能把万客来抢走？核心基线模型在我手里，境外资本的钱已经进了云帆的托管账户，今天午夜交割完，万客来的线下牌照就是我的，你爹十年心血，到头来还是给我做了嫁衣。”

苏哲指尖敲了敲亮着的LLM界面，屏幕上正滚动着王堔最近一周的行踪轨迹，开口道：“你别急，午夜还早，咱们谁能拿到东西，还不一定。”

挂了电话，林溪刚好查完资金流水推门进来，把平板放在桌面对准苏哲：“对上了，境外资本的托管账户确实已经到位，就等拿到模型授权就打款。陈曦那里我留了人盯着，搜出了她藏的备用机，刚看完聊天记录——他们拿到基线模型，直接拿去微调自己的通用LLM，比从零开始训练至少省两年时间，这才是他们盯上万客来的真正目的，根本不是什么本地生活市场。”

苏哲揉了揉眉心，重生回来他比任何人都清楚这点。前世他到死才幡然醒悟，王堔勾结境外资本，从一开始要的就不是万客来的生意，是他父亲苏文远花了整整五年，用百亿级本地生活数据训练出来的核心基线模型——那玩意儿在2019年，是多少资本抢破头都拿不到的宝贝。他靠着重生的信息差提前半年训出自己的私有小LLM，就是为了今天能抢在王堔前面把东西拿回来。

苏哲合上笔记本站起身，开口道：“走吧，去机房。张广涛，你带路，拿到模型校验完，咱们正好给王堔送份大礼。”

四十分钟后，车停在了临江市郊外IDC机房的院子里，门卫早就被张广涛打过招呼，几个人顺着安全通道走到三楼第三排机柜前。张广涛挪开最边上堆满纸箱的挡板，三台落满灰的离线服务器安安静静立在那里，插电开机，蓝色的电源灯一点点亮了起来。

苏哲把加密移动硬盘插上去，调出私有LLM做哈希校验，屏幕上的进度条一点点往前爬。张广涛站在他身后，手悄悄扣紧了口袋里的手机，林溪靠在门口放风，掏出烟给张广涛递了一根。

进度条走到一百的时候，绿色的“校验通过”四个字跳了出来，LLM明确标注，这就是2018年底苏文远最后一次更新的核心基线模型，分毫不差。

苏哲刚松了口气，口袋里的手机突然震了一下，是私有LLM自动推送的后台预警。他点进预警弹窗，瞳孔瞬间缩紧——LLM扫描服务器后台时，扫出了一个隐藏极深的后门程序，这个后门不到一个小时前，刚刚主动向一个备案在境外的IP发送了完整的模型哈希值和机房定位。而之前LLM溯源的那笔三百万进项，最终收款人信息刚刚完成解密，签字确认人，就是站在他身后的张广涛。

机房楼下突然传来汽车引擎熄灭的轰鸣，紧接着是关车门的闷响，杂乱的脚步声顺着楼梯往三楼涌上来，最前面那个熟悉的声音，隔着门板飘进来，清清楚楚落进了房间里每个人的耳朵。

“苏哲，我知道你在里面，把硬盘交出来，我给你个痛快，别跟你爹一样，非要落得尸骨无存。”
