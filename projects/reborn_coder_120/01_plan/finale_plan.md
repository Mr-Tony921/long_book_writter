# 完结篇大纲：151-202

## 三阶段划分

### Stage A · 151-180（30 章）主线清算
- **目标**：把已埋设伏笔、已出现反派、已建立关系**逐项回收**，逐步走向终局。
- **硬约束**：
  - **不再引入未铺垫的全新对手或全新组织**（前 150 章已经够多了）
  - 每章必须回收一条已有线索（已落网反派的法律后续、未结伏笔、关系节点）
  - 时间线继续向前
  - 父亲苏振海的状态需在本阶段中段开始明确好转
- **末段需达成**：所有中层势力清算完毕，仅剩最高层 boss 与跨境收尾未结

### Stage B · 181-192（12 章）关系与事业收束
- **目标**：主线已基本收束，重点处理人物关系与事业线。
- **硬约束**：
  - 节奏放缓，允许「无新反派」的章节
  - 重点回收：
    - LLM 辅助工具的最终归宿（开源/移交/自研团队）
    - 苏振海全面康复并归位（家庭团聚）
    - 王堔/陈曦/张广涛等已落网反派的法律审判进度
    - 苏哲与林溪的关系明确升温（未点破）
    - 苏哲与陈曦从对立到和解的边界（陈曦本是内鬼但有家人胁迫的隐情）
  - 不强求「不可逆推进」
  - 情感张力优先于动作张力

### Stage C · 193-202（10 章）高潮与完结
- **节奏**：
  - 193-197（5 章）：冲击至高潮顶点。最高层 boss（最终幕后）显形并决一胜负，跨境势力被一锤定音。苏振江最终判决落地。
  - 198-201（4 章）：每位关键人物的明确去向：
    - 苏哲：完成跨界整合，事业站稳
    - 林溪：与苏哲关系定调（不点破未来，留白）
    - 苏振海：晚年归位
    - 张广涛、王堔、陈曦：法律收束
    - 父亲十年旧案：彻底盖棺
  - 202（1 章）：完结章。以日常画面收尾，**末段留一处不影响主线的情感钩子** ——苏哲/林溪未完全说破的未来：例如林溪的一个动作或一句未说完的话，让读者自己想象。
- **硬约束**：
  - 末章不能引入新冲突
  - 末章必须有「闭合感」：所有 must_mention 伏笔都已回收
  - 字数仍 ≥2200
